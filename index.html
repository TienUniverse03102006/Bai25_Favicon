Hướng dẫn tạo và tối ưu cho đa thiết bị trong lập trình web

Favicon trong HTML là biểu tượng nhỏ đại diện cho website. Nó thường xuất hiện ở:

  1. Tab trình duyệt
  2. Thanh bookmark
  3. Lịch sử trình duyệt
  4. Màn hình chính (nếu web được ghim trên mobile)

Cách thêm favicon vào HTML

  1. Trong khối head -> gõ link: favicon -> dán link favicon muốn vào href
  2. Vì có nhiều kích cõ khác nhau: nên tạo bộ hình ảnh tương thích với thiết bị:
    1. Lên web tìm make favicon online-> chọn 1 trang web để tạo favicon -> ném ảnh đã lưu lên để tạo favicon
    2. Sau khi xong dowload về-> vào html nhấn chuột phải -> reveal in file explore
    3. Ném file vừa tải về vào trong -> extract all -> đổi tên file ví dụ: iconUni
    4. Vào lại trang web vừa nãy tạo favicon -> sao chép đoạn code về link: favicon -> paste vào khối head
    5. Thay đổi tất cả đường dẫn ảnh href="/ về đường dẫn tương đối href="./iconUni
